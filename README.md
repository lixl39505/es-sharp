# es-sharp

collection of commonly used js functions.

## Installation

In a browser:

```html
<script src="es-sharp.min.js"></script>
```

Using npm:

```shell
$ npm i es-sharp
```

In Node.js:

```js
// load the full build
var _ = require('es-sharp')
// or load on demand
var fp = require('es-sharp/lib/fp')
var qs = require('es-sharp/lib/string/qs')
var event = require('es-sharp/lib/event')
...
```
